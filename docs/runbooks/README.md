# Runbooks

Documentos de referência consultável durante operação. Não são tutoriais progressivos; são listas de comandos organizadas por tarefa para uso cotidiano e diagnóstico.

Para reproduzir uma configuração nova, use um [guide](../guides/). Para entender os conceitos por trás de uma técnica, leia a [lesson](../lessons/) correspondente. Para o registro do que a IA + operador efetivamente fizeram em operações reais, veja [ai-ops](../ai-ops/). Para o porquê de uma decisão arquitetural, veja o [ADR](../adr/) relevante.
