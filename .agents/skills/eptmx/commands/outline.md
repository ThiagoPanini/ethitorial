# outline — só a estrutura, sem a prosa

Para quando o autor quer **pensar a estrutura antes** de comprometer a prosa. Mesma entrevista
e mesma resolução de destino/tags do [write](write.md), mas você **para no esqueleto aprovado**.
Útil para validar o arco de um post denso, ou para fatiar uma série antes de escrever.

```
Entrevistar → Resolver destino → Resolver tags → Entregar o esqueleto → (opcional) seguir para write
```

## 1. Entrevistar

Igual ao [write §1](write.md), mas você pode ir mais raso nos **insumos concretos** (comandos,
código, números): aqui o foco é tese, público, arco e seções, não o material de cada passo.
Insista na **tese** — um outline sem ângulo vira sumário morto.

## 2. Resolver destino e tags

Faça a resolução de destino ([write §2](write.md)) e de tags ([write §3](write.md)) — **propor
antes de gravar**, sempre. Você *pode* já gravar o plumbing aprovado (Section/Source/tags) se o
autor confirmar, para que o `write` posterior caia em terreno pronto. Mas **não crie o `.mdx`**.

## 3. Entregar o esqueleto

Produza um plano enxuto, não prosa:

- **Título provisório** e **slug** candidato.
- **Tese** em uma frase (a única ideia que o leitor leva).
- **Destino**: caminho final do `.mdx` (mesmo que o arquivo ainda não exista) + qualquer
  plumbing necessário.
- **Tags**: existentes + adições propostas.
- **Esqueleto**: a lista de headings na ordem do arco, **cada um com uma linha** dizendo o que
  aquela seção entrega (a promessa concreta — DNA §1.2 do STYLE). Para tutorial, deixe o arco
  pré-requisitos → passos → validação → armadilhas visível.
- **Insumos a coletar**: o que falta levantar (comandos, prints, versões) antes de escrever.

Não escreva parágrafos de conteúdo. O objetivo é um mapa que o autor aprova ou ajusta.

## 4. Seguir para write (opcional)

Feche perguntando se ele quer que você **escreva o draft agora** a partir deste esqueleto. Se
sim, siga o [write](write.md) a partir do §4 (Escrever o draft), reaproveitando tudo que já foi
decidido — sem re-entrevistar o que já está resolvido.
